## Getting Started

### Prerequisites (OSX)
- [x] Go installed | go1.20.2 darwin/amd64

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/mzhar91/golang-example-edts.git
   ```

### Run Application
1. Run go
   ```sh
   cd {to each folder} && go run main.go
   ```
